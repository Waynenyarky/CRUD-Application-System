Welcome to the CRUD Application Help Page

Overview

This help page is designed to guide you through the basic functionality of our CRUD application.
The application allows you to Create, Read, Update, and Delete records in a database.
Below are the main operations you can perform and instructions on how to use them effectively.

Creating a Record

  1. Navigate to the "Create" section of the application.
  2. Fill in the required fields with the necessary information for the new record.
  3. Click the "Submit" button to create the record.
Reading a Record
  1. Go to the "Read" section of the application
  2. Use the search or filter options to locate the record you want to view.
  3. Click on the record to display its details.

Updating a Record

  1. Access the "Update" section of the application.
  2. Find the record you wish to update using search or filtering.
  3. Click on the record to open the edit form.
  4. Make the desired changes to the record.
  5. Click the "Save" button to update the record.

Deleting a Record

  1. In the "Delete" section, locate the record you want to delete.
  2. Click on the record to select it.
  3. Confirm the deletion action when prompted.
